# PRIME XI - Frontend Dashboard 🏆

**PRIME XI** es una plataforma de análisis avanzado y optimización para la Fantasy Premier League, diseñada para ofrecer una ventaja competitiva mediante una interfaz de élite.

## 🎨 El Objetivo: "Midnight Precision UI"
Nuestro norte absoluto es el diseño premium: un dashboard en dark mode (Midnight Navy), con efectos de glassmorphism, visualización de campo de juego y gráficas analíticas de primer nivel.

## 🛠️ Stack Tecnológico
- **Framework:** Next.js (App Router)
- **Estilos:** Tailwind CSS
- **Animaciones:** Framer Motion
- **Iconografía:** Lucide React / Tabler Icons
