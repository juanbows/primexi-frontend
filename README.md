# PRIME XI - Fantasy Premier League IA Dashboard 🏆

**PRIME XI** es una plataforma de análisis avanzado y optimización estratégica para la **Fantasy Premier League (FPL)**. Diseñada para mánagers que buscan la perfección mediante una interfaz de élite impulsada por inteligencia de datos.

---

## 🎨 El Objetivo: "Midnight Precision UI"
Nuestro norte absoluto es el diseño premium: un ecosistema digital en **Dark Mode (Midnight Navy)**, con efectos de glassmorphism, visualización táctica del campo de juego (Pitch View) y gráficas analíticas de primer nivel.

## 🛠️ Stack Tecnológico
- **Frontend Framework:** Next.js (App Router)
- **Database / Backend:** Supabase (PostgreSQL)
- **Styling:** Tailwind CSS (Elite Theme)
- **Animations:** Framer Motion / Canvas API (Partículas)
- **Iconography:** Lucide React / Tabler Icons

---

## 🏗️ Arquitectura del Sistema
El proyecto se divide en módulos técnicos coordinados por **Bottby** (Asistente Técnico):

1. **Data Ingestion:** Conexión a la API oficial de la FPL para extracción de estadísticas en tiempo real (xG, xA, ICT Index).
2. **Database Schema:** 6 tablas relacionales en Supabase (`elements`, `teams`, `events`, `fixtures`, `types`, `history`).
3. **AI Engine:** Algoritmo de optimización que predice el mejor 11 semanal bajo restricciones de presupuesto y formación.
4. **UI/UX Layer:** Interfaz interactiva con efectos 3D, cursores personalizados y feedback visual avanzado.
