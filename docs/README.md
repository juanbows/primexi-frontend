# PRIME XI - Inteligencia Artificial para FPL

PRIME XI es un asistente avanzado para Fantasy Premier League (FPL) que utiliza optimización matemática y análisis de datos para ayudar a los mánagers a dominar sus ligas.

## Características Principales
- 🚀 **Optimizador Multi-Jornada:** Visión a 3 semanas para planificación estratégica (Solo Premium).
- 📊 **xP Heurístico:** Cálculo de puntos esperados basado en forma, dificultad de encuentros y probabilidad de juego.
- ⚡ **API de Alto Rendimiento:** Backend construido con FastAPI para respuestas instantáneas.
- 🌙 **Diseño Midnight Precision:** Interfaz premium, oscura y profesional.

## Estructura del Proyecto
- `/backend`: Motor de optimización, scrapers y API.
- `/frontend`: Aplicación web moderna (Next.js).
- `/docs`: Documentación detallada del proyecto.

## Inicio Rápido (Backend)
1. Instalar dependencias: `pip install -r requirements.txt`
2. Ejecutar servidor: `uvicorn app.main:app --reload`
