# Sobre PRIME XI

## Nuestra Misión
Empoderar a los mánagers de la Fantasy Premier League con herramientas de nivel profesional impulsadas por IA. Queremos democratizar la ventaja competitiva: que cualquier mánager, ya sea casual o experto, pueda tomar decisiones estratégicas basadas en datos reales y no en corazonadas.

## ¿Qué es PRIME XI?
PRIME XI es el asistente definitivo para FPL, con análisis predictivo, algoritmos de transferencia en tiempo real y selecciones de capitán basadas en datos.

- **Temporada activa:** 24/25
- **Usuarios en la plataforma:** +10,000 mánagers activos
- **Ranking proyectado para nuestros usuarios:** Top 1%
- **Costo de entrada:** £0 (sin tarjeta requerida en el periodo de prueba)

## Stack Tecnológico del Proyecto
- **Frontend:** HTML5, CSS3 (Tailwind CSS via CDN), JavaScript vanilla
- **Fuentes:** Space Grotesk + Inter (Google Fonts)
- **Animaciones:** CSS Keyframes + IntersectionObserver API
- **Video:** YouTube Iframe API
- **Control de versiones:** Git + GitHub (`juanbows/primexi-frontend`)
